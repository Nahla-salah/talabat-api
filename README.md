# 🛒 Talabat API Clone

## 💻 Tech Stack
- **Framework:** .NET 8 Web API
- **Architecture:** Onion Architecture (Clean Architecture)
- **Patterns:** Repository Pattern, Unit of Work
- **Database:** SQL Server with Entity Framework Core
- **Identity:** JWT Authentication & Authorization

## 🏗️ Project Structure
- **Core (Domain):** Entities & Contracts
- **Infrastructure (Persistence):** Data Context, Migrations, & Repository Implementation
- **Application (Services):** Business Logic
- **Presentation (API):** Controllers & Middlewares
